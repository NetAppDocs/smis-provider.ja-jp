= CIMOM コマンド
:icons: font
:imagesdir: ./media/


[role="lead"]
「 cimconfig 」コマンドを使用して、 HTTP および HTTPS のイネーブル化とディセーブル化、 HTTP および HTTPS ポート番号の変更などの CIMOM 設定を行うことができます。
